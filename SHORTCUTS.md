# VibeCut Keyboard Shortcuts & Gestures

## Playback Controls

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |

## Timeline Navigation

| Key | Action |
|-----|--------|
| `←` Left Arrow | Move back one frame |
| `→` Right Arrow | Move forward one frame |
| `↑` Up Arrow | Go to beginning of current clip (press again for previous clip) |
| `↓` Down Arrow | Go to end of current clip (press again for next clip) |

## Editing

| Key | Action |
|-----|--------|
| `B` | Split clip at playhead position |

## Timeline Zoom & Scroll

| Gesture | Action |
|---------|--------|
| Mouse Wheel | Zoom in/out (centered on cursor) |
| Shift + Mouse Wheel | Scroll timeline left/right |
| Horizontal Scroll (touchpad) | Scroll timeline left/right |
| Middle Mouse Drag | Pan timeline |
| Alt + Left Mouse Drag | Pan timeline (alternative) |
| Drag Scrollbar | Scroll timeline |
| Click Scrollbar Track | Jump to position |

## Timeline Clip Editing

| Gesture | Action |
|---------|--------|
| Drag Left Handle | Trim clip start |
| Drag Right Handle | Trim clip end |
| Click Timeline | Move playhead to position |

## Zoom Limits

- **Minimum zoom**: 0.25x (see 4x the content)
- **Maximum zoom**: Dynamic, calculated so 1 frame ≈ 50 pixels

## Notes

- Clips have a minimum duration of 1 frame
- Clips can become very small when zoomed out - zoom in to see/edit them
- Timeline scale is based on source video duration for consistent pixel sizing
